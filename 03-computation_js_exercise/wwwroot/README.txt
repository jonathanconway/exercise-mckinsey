
README
======

• You can execute my solution by opening 'index.html' in your browser.

• Unit tests are in 'js/tests.js'. You can execute them by loading 'index.html'
  in your browser, opening the console, and executing 'testCountSubsequences()'.

• Since 1,000 characters of pi are too long a sequence to verify manually, I
  created a smaller test string, 'the cat sat on the mat', and a visual matrix
  in order to verify with my own eyes that the function was running correctly.
